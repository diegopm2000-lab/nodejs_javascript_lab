# Cucumber HelloWorld in typescript

This is a minimal example of cucumber based on the example from npm @cucumber/cucumber repository.

## 1. Install the program

Execute this:

```shell
npm i
```

## 2. Run the bdd tests

Execute this:

```shell
npm run test
```
